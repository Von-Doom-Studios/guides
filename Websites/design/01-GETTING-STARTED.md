# Getting Started — Floki Website Design

Welcome to the Floki website design system. This folder is the single source of truth for all design decisions, asset creation, and AI-assisted content generation.

---

## Who This Is For

Anyone working on the visual side of the Floki website — whether you're generating images, writing copy, building out new sections, or handing assets to the dev team.

## The Process in Brief

1. **Find references** — use Pinterest to collect images styled the way we want (see `04-REFERENCE-SYSTEM.md`)
2. **Generate assets** — use those references to write Midjourney prompts featuring Floki products (see `03-AI-DESIGN-WORKFLOW.md`)
3. **Generate copy** — use Claude or ChatGPT with brand voice guidelines
4. **Review** — check everything against the Style Guide before handing off
5. **Hand off to dev** — drop assets in the right folder and post in #dev with context

---

## Ground Rules

- **Always check the Style Guide first.** Colors, fonts, and button styles are locked. Don't invent new ones.
- **AI output is a starting point, not a final product.** Review everything before it goes to the site.
- **Name files correctly.** See asset naming conventions in the Style Guide.
- **Never paste raw code in Discord.** Post in the right repo or hand off via #dev.
