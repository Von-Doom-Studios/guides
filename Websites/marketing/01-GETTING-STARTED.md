# Marketing — Getting Started

This folder is owned by **VD-Marketing**. All marketing strategy, brand guidance, and copy work for Floki, VonDoom Studios, and Floating Objects Studios lives here.

## Folder Structure

| File | Purpose |
|---|---|
| `01-GETTING-STARTED.md` | This file |
| `02-BRAND-VOICE.md` | Brand voice, tone rules, and writing guidelines |
| `03-WEBSITE-COPY.md` | Working copy for all Floki website pages |
| `04-CAMPAIGNS.md` | Active and planned campaigns |
| `05-CONTENT-STRATEGY.md` | Content pillars, formats, and cadence |
| `06-CHANNELS.md` | Channel strategy (social, email, paid, etc.) |
| `07-ANALYTICS.md` | KPIs, reporting, and measurement |

## Workflow

- Website copy is developed and tracked in `03-WEBSITE-COPY.md`
- When a page is complete, hand off to VD-Dev in #dev for implementation
- Brand voice in `02-BRAND-VOICE.md` governs all copy across every channel
