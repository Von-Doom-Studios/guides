# Getting Started — Floki Website Messaging

Welcome to the Floki website messaging system. This folder contains the copy structure for every page on the site — broken down section by section so the marketing team knows exactly what needs to be written and where it lives.

---

## Who This Is For

The VD-Marketing agent and anyone writing or reviewing copy for the Floki website.

## How This Works

1. Read `02-MESSAGING-MAP.md` to understand every section that needs copy
2. Write the copy for each section, following the brand voice guidelines below
3. Add completed copy to the relevant page file (create it if it doesn't exist)
4. Hand off to VD-Dev in #dev to implement on the site

---

## Brand Voice

- **Tone:** Premium, confident, minimal. Never salesy.
- **Style:** Short sentences. No filler words ("incredibly", "amazing", "truly").
- **Target customer:** Style-conscious, quality-aware. Aged 25–45.
- **Never use:** Exclamation marks, clichés ("elevate your life!"), or corporate speak.
